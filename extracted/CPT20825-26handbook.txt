CPT208 Human-Centric Computing
Section A: Basic Information
1. Brief Introduction to the Module
Human-Centric Computing is a level one module that provides students with up-to-date guidelines, concepts and
models for designing and evaluating interactive systems.
Students will learn theories and methodologies in designing and implementing graphical user interfaces through
practical tools.

Section B: Module Information
1. Educational Aims
This module aims to:
Understand design process in computing systems
Cover requirement gathering, design, prototyping, evaluation
Apply interaction design methods in practical projects
Evaluate usability and user experience
2. Learning Outcomes
Students will be able to:
A. Apply user-centered design approach emphasizing equality, diversity, and inclusion
B. Create and deploy interactive systems with legal, ethical, and professional considerations
C. Evaluate usability and user experience using established criteria
